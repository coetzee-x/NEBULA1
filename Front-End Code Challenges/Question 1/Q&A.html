Question 1.1

What will the colour of both div elements be?

Answer 1.1

  • The div with id firstDiv will have a background of red.
  • The div with id secondDiv will have a background of orange.

Question 1.2

How would you dynamically target firstDiv and make it's colour pink? (provide code snippet).

Answer 1.2

document.getElementsByTagName("body")[0].children[0].style.background = "pink";

Question 1.3

How would you dynamically target the secondDiv and add the yellow-card class to its class list? (provide code snippet).

Answer 1.3

document.getElementsByTagName("body")[0].children[1].classList.add("yellow-card");
