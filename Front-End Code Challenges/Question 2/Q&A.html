Question 2.1

Why will the following function return true?

function compareIt(num1, num2){ return num1 == num2; }

Answer 2.1

Becuase when '==' is used to compare two values, the data type is ignored.

Question 2.2

How could one change this function so that the data types are checked as well as values?

Answer 2.2

function compareIt(num1, num2){ return num1 === num2; }

Comparing a string value with an int.
